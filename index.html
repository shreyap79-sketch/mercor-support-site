Mercor — Candidate Support Answers

Roleplay responses + Application Debugging steps + SQL answers

Roleplay Answers

Q1. I-983 signature

Mercor doesn’t sign I-983 forms. Send it to your employer of record (the company that hired you). If unsure, check your Contracts tab or onboarding emails. For help: support@mercor.com.

Q2. It’s Thursday; no Wednesday payment

  • Payouts process Wednesday for approved hours from the prior week.
  • Banks can take 1–3 business days. If nothing by Friday EOD, verify payout details and contact support.

Q3. /upload-resume 404

That URL is outdated. Upload your resume from the Applications tab after login. If issues persist, clear cache/try another browser or ask support for a fresh link.

Q4. Referred 40 people; when do I get paid?

  • Rewards pay only after a referred candidate is hired and completes eligibility (e.g., minimum time worked).
  • Track status and payout in the Referrals dashboard.

Application Debugging (Chrome DevTools)

Q1. Find interviewId & status

  1. Open Interviews → F12 → Network → filter interview.
  2. Click “Talent Success Interview” → open request → Response → read id/status.

Q2. Resume → Work Experience: endpoint + method

  1. Edit Work Experience → Save while Network is open.
  2. Endpoint will include experience/resume. Using PATCH (partial update) is ideal; PUT is OK if replacing entire object; POST only for create/upsert.

Q3. When was the job listing created?

  1. Network filter listing and read createdAt in the response, or search __NEXT_DATA__.
  2. Convert in Console: new Date("2025-09-04T12:34:56.789Z").toString()

SQL Queries

Q1. Users who applied to “Mercor” listings

SELECT DISTINCT
  u.UserId,
  u.Email
FROM Applications a
JOIN Listings    l ON l.ListingUID = a.ListingVersionId
JOIN Company     c ON c.CompanyId  = l.CompanyId
JOIN MercorUsers u ON u.UserId     = a.UserId
WHERE c.Name = 'Mercor';

Q2. Avg min/max rate (USD) for “Mercor” listings

SELECT
  AVG(l.RateMin) / 100.0 AS AvgMinRateUSD,
  AVG(l.RateMax) / 100.0 AS AvgMaxRateUSD
FROM Listings l
JOIN Company  c ON c.CompanyId = l.CompanyId
WHERE c.Name = 'Mercor';

Rates stored in cents → divide by 100.
